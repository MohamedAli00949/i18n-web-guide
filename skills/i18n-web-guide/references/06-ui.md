# UI & I18n

## HTML Root
```html
<html lang="ar" dir="rtl">
<html lang="en" dir="ltr">
```
Radix UI: `<DirectionProvider dir="rtl">`. shadcn/ui: [RTL guide](https://ui.shadcn.com/docs/rtl/).

## Writing Systems
LTR: Latin, Indic · RTL: Arabic, Persian, Hebrew · Bidi: mixed · Vertical: Japanese, Chinese, Mongolian

## `dir` Attribute
Values: `ltr` / `rtl` / `auto` (detects from first strong char)
- Use `dir="auto"` per paragraph in text editors for mixed-direction UGC
- Style with `:dir()` (semantic) or `[dir=rtl]` (explicit attr only)

## `unicode-bidi` CSS
`normal` · `embed` · `isolate` (fence from surrounding bidi) · `bidi-override` (force direction) · `plaintext` (ignore CSS dir)
⚠️ Prefer HTML `dir` over CSS `direction`/`unicode-bidi` in HTML docs.

## `writing-mode`
`horizontal-tb` (default) · `vertical-rl` (Han/Japanese) · `vertical-lr` (Mongolian) · `sideways-rl/lr`

## Typography
- Load fonts per locale (performance) or include all (auto-detect, heavier)
- ⚠️ **Never `letter-spacing` on Arabic/Persian** — breaks connected letterforms (CSS spec limitation)
- `line-height` affects readability per script

## Logical CSS (required for RTL auto-flip)
| Physical | Logical |
|---|---|
| `margin-left` | `margin-inline-start` |
| `margin-right` | `margin-inline-end` |
| `padding-left` | `padding-inline-start` |
| `text-align: left` | `text-align: start` |
| `border-right` | `border-inline-end` |

Tailwind: `ps-*`, `pe-*`, `ms-*`, `me-*`, `text-start`

## Form Fields
- **Date picker**: configure locale + timezone + calendar system (all three)
- **Text editor**: `Intl.Segmenter` for char counts, `dir="auto"` per paragraph, logical CSS throughout
- **Vertical fields**: `writing-mode` + logical properties

## Translate These Attributes
`title`, `alt`, `placeholder`, `aria-label`, `aria-description`, `data-*` (if displayed)
