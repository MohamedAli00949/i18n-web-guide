# Tools

## Libraries

| Library | Best for | Notes |
|---|---|---|
| [lingui](https://lingui.dev) | JS/React/Vue/Next.js | Best DX. Macro-based extraction, lazy load, AI translation. RSC support since v4.10. |
| [next-intl](https://next-intl.dev/) | Next.js | Gold standard. Server + Client Components. Built-in routing. |
| [react-i18next](https://react.i18next.com/) | React (Vite/CRA) | Most popular. Batteries included. |
| [formatJS/react-intl](https://formatjs.github.io/) | React (enterprise) | Strict ICU/web standards. Built by Yahoo. |
| [vue-i18n](https://vue-i18n.intlify.dev/) | Vue.js | Official Vue plugin. |
| [@angular/localize](https://v17.angular.io/api/localize) | Angular | Compile-time. Zero runtime overhead. |
| [Transloco](https://github.com/jsverse/transloco) | Angular (alt) | Runtime switching, better lazy-loading. |

## ESLint
- [eslint-plugin-i18n-checker](https://www.npmjs.com/package/eslint-plugin-i18n-checker) — missing key detection
- [eslint-plugin-rtl-friendly](https://www.npmjs.com/package/eslint-plugin-rtl-friendly) — enforce logical CSS
